# TechStore Landing Page

## Descripción
Landing Page promocional para una tienda online de periféricos tecnológicos.

## Tecnologías
- HTML5
- CSS3
- GitHub

## Prompt usado en IA
Actúa como diseñador UI profesional.

Crea únicamente el CSS base para una Landing Page moderna de e-commerce llamada TechStore.

Reglas:
- Usa CSS Grid para un catálogo responsivo.
- Móvil: 1 columna.
- Tablet: 2 columnas.
- PC: 4 columnas.
- Diseña botones atractivos con hover.
- Usa colores modernos y contraste AAA para precios.

## Principio UX aplicado
Para los botones "Añadir al carrito" se utilizó jerarquía visual y affordance:
- Color contrastante
- Tamaño grande
- Bordes redondeados
- Hover visual para feedback inmediato
