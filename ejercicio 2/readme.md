# Actividad 2 - Landing Page Cafetería

## Explicación técnica

se usa  Flexbox para cambiar la dirección entre imagen y texto porque facilita
controlar la alineación de elementos en un sentido.
Flexbox permite cambiar fácilmente fila y columnas 
Grid es mejor para layouts más complejos de dos dimensiones mientras que
Flexbox resulta más simple y eficiente para este componente.
